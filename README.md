# HelloOS
 Простая операционная система для x86 процессоров.<br/>
 (qemu-system-i386 -kernel bin/kernel.bin)
 
# Сделано
* Текстовый display драйвер
* Keyboard PS2 драйвер
* malloc (+ free)
* itoa (кривой), strlen
* Работа с портами (ports.c)
